AI Threat Detection System

I built this project to explore how computer vision can be used in a simple security system. It uses Python along with OpenCV and Tkinter to monitor real-time activity through a webcam and detect possible threats.

The system performs face detection, full-body detection, and motion analysis at the same time. By comparing consecutive frames, it identifies movement and decides whether the activity is normal or suspicious. When something unusual is detected, it highlights it directly on the screen.

I also added a basic login system and a dashboard interface using Tkinter, so the application feels more complete and user-friendly instead of just being a script.

Features.
Real-time face detection.  
Full body detection.  
Motion detection.  
Suspicious activity indication.  
Simple login system with dashboard.  

Technologies Used.
Python.  
OpenCV.  
NumPy.  
Tkinter.
